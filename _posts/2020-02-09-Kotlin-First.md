## 표현식(Expression)  
- 하나의 값으로 수렵하는 수식 뭉치  
- 값을 반환한다.  
- 변수에 값을 할당하는데 쓰일 수 있다.  
- 표현식의 결과에는 타입이 존재한다.  
표현식의 타입은 중요! 변수에 값을 저장하려면 변수의 타입과 표현식의 타입이 일치해야 한다.  
ex. val somthing: Int = 결과 타입이 Int 인 표현식
